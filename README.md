# AI Chatbot

This project is a simple and flexible AI Chatbot built using Python.
It includes multiple chatbot scripts that interact with different AI models and provide responses based on user input.
The repository is designed for experimenting with chatbot logic, API calls, and conversational AI behavior.

## Overview
- This chatbot project allows you to:
- Communicate with AI models using Python.
- Test different chatbot versions through separate .py files.
- Experiment with prompt design, conversation flow, and API usage.
- Run chatbot scripts directly in the terminal.
- Store API keys securely using a .env file.
- Explore model performance using a Jupyter Notebook.

## Features
- Python-Based Chatbot
Simple to run and modify. Easy to extend for learning or development.

- Multiple Implementations
Each file offers a different style of chatbot logic or model integration.

- API Integration
Supports AI models through API keys stored in .env.

- Jupyter Notebook Support
new.ipynb allows testing ideas, prompts, and responses interactively.

## Requirements
Install dependencies):
pip install python-dotenv requests openai

## How to Run
Add your API key
Create a .env file:
API_KEY=your_api_key_here

## Run any chatbot file
Example:
python deepseek.py
